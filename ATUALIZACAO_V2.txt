🚀 COMO USAR A NOVA FUNCIONALIDADE
================================================================================

1. FAÇA LOGIN
   └─ Acesse seu site normalmente
   └─ Faça login com suas credenciais

2. CLIQUE EM "🔍 BUSCAR"
   └─ Na navbar (barra superior)
   └─ Nova aba aparecerá

3. DIGITE ALGO
   └─ Nome do fornecedor
   └─ Tipo de peça
   └─ Palavra-chave

4. CLIQUE "BUSCAR"
   └─ Aguarde 2-3 segundos
   └─ Resultados aparecem

5. EXPLORE RESULTADOS
   └─ Seção 1: Fornecedores já cadastrados
   └─ Seção 2: Resultados da web

6. ADICIONE FORNECEDORES (Opcional)
   └─ Clique "Adicionar" em resultados da web
   └─ Fornecedor salvo automaticamente
   └─ Próxima busca, aparece em "Cadastrados"

⚙️ CONFIGURAÇÃO NECESSÁRIA
================================================================================

NECESSÁRIO (Já feito se você seguiu o tutorial anterior):
  ✅ Variáveis Supabase em .env.local
  ✅ Banco de dados Supabase configurado

PARA BANCO ATUALIZADO (Recomendado):
  1. Abra SQL Editor do Supabase
  2. Execute código em ATUALIZACAO_BANCO.txt
  3. Pronto! Suporta tags agora

PARA BRAVE SEARCH (Opcional):
  1. Visite https://api.search.brave.com/
  2. Crie conta (free)
  3. Copie sua API Key
  4. Adicione em .env.local:
     VITE_BRAVE_API_KEY=sua_chave

IMPORTANTE: Sistema funciona PERFEITAMENTE sem Brave!
           DuckDuckGo é completamente gratuito.

🔒 SEGURANÇA
================================================================================

✅ API Keys nunca são expostas
   └─ Brave API Key fica em .env.local (não commitada)
   └─ No Netlify, fica em variáveis secretas

✅ DuckDuckGo é anônimo
   └─ Sem rastreamento
   └─ Sem privacidade invadida

✅ Supabase tem RLS
   └─ Apenas usuários autenticados buscam
   └─ Cada usuário vê seus próprios dados

✅ URLs são validadas
   └─ Devem começar com https://
   └─ Protege contra sites maliciosos

================================================================================
🐛 TROUBLESHOOTING
================================================================================

❌ "Nenhum resultado encontrado"
   → Tente buscas diferentes
   → Verifique sua internet
   → Recarregue a página (F5)

❌ "Brave não está funcionando"
   → É opcional! DuckDuckGo funciona sozinho
   → Se quiser Brave: adicione API Key em .env.local
   → Reinicie: npm run dev

❌ "Não consigo adicionar fornecedor"
   → Verifique se já está cadastrado
   → Abra F12 → Console
   → Procure por erros

✅ Para mais dúvidas, leia TUTORIAL_BUSCA_AVANCADA.txt
