QUÊ? Seu sistema agora tem BUSCA em 3 fontes simultaneamente!

✅ Banco interno (Supabase)
✅ Web (DuckDuckGo - gratuito!)
✅ Web (Brave - opcional, com API key)

ONDE? Nova aba "🔍 Buscar" na navbar (barra superior)

COMO? 
  1. Faça login
  2. Clique em "🔍 Buscar"
  3. Digite o que procura
  4. Clique "Buscar"
  5. Veja resultados em 2 seções
  6. Clique "Adicionar" para salvar fornecedores

QUANDO? Agora mesmo! Já está em produção.

POR QUÊ? Para você encontrar fornecedores em um único lugar!

⚡ 5 PASSOS PARA COMEÇAR AGORA
================================================================================

1. RODAR NO SEU PC
   $ npm run dev
   → Abre em http://localhost:5173

2. FAZER LOGIN
   → Coloque seu email e senha

3. CLICAR EM "🔍 BUSCAR"
   → Na barra superior (navbar)

4. TESTAR A BUSCA
   → Digite "Rufato" ou qualquer coisa
   → Clique "Buscar"
   → Veja resultados aparecerem!

5. ADICIONAR FORNECEDOR (Opcional)
   → Clique "Adicionar" em um resultado
   → Fornecedor salvo automaticamente!

PRONTO! Você está usando v2.0! 🎉

❓ PERGUNTAS FREQUENTES
================================================================================

P: "A busca funciona sem configurar Brave?"
R: ✅ SIM! DuckDuckGo sozinho já é excelente e gratuito!

P: "Quanto custa usar DuckDuckGo?"
R: ✅ NADA! É 100% gratuito e sem limite de buscas!

P: "Preciso fazer alguma coisa para começar a usar?"
R: ✅ NÃO! Já está tudo integrado. Faça login e teste!

P: "Onde aparece a aba de busca?"
R: Na barra superior (navbar), ao lado de "📦 Fornecedores"

P: "Posso adicionar fornecedores sem cadastrar antes?"
R: ✅ SIM! A busca encontra na web e você adiciona com 1 clique!

P: "Os fornecedores adicionados ficam salvos?"
R: ✅ SIM! No Supabase, persistente e sincronizado!

P: "Funciona em produção (Netlify)?"
R: ✅ SIM! Já foi deployado automaticamente!

P: "Preciso de API keys?"
R: ❌ NÃO para DuckDuckGo! Sim para Brave (mas é opcional).

P: "Qual é o limite de buscas?"
R: DuckDuckGo = ilimitado. Brave = 300/mês (free). Supabase = seu plano.

P: "E se eu estiver offline?"
R: Busca na web não funciona. Busca local (banco) funciona se tiver dados.

🐛 ALGO ESTRANHO? VEJA AQUI
================================================================================

❌ "Aba 'Buscar' não aparece"
   → npm run dev está rodando?
   → Recarregue F5
   → Está logado?

❌ "Nenhum resultado"
   → Tente outra busca
   → Verifique internet
   → Recarregue F5

❌ "Erro ao adicionar"
   → Já está cadastrado?
   → URL válida?
   → Abra F12 → Console → veja erro

❌ "Brave não funciona"
   → É OPCIONAL! Funciona sem ele!
   → DuckDuckGo sozinho é suficiente
