🚀 COMO COMEÇAR
================================================================================

PASSO 1: RODAR LOCALMENTE

$ cd C:\Users\Administrator\Downloads\plataformas\portal-autopeças
$ npm install (se não tiver feito)
$ npm run dev

PASSO 2: TESTAR NO NAVEGADOR

→ Acesse http://localhost:5173
→ Faça login
→ Na navbar, clique em "🔍 Buscar"
→ Digite algo (ex: "Rufato")
→ Clique "Buscar"
→ Veja os resultados! 🎉

PASSO 3: ADICIONAR FORNECEDORES (Opcional)

→ Nos resultados da web, clique "Adicionar"
→ Fornecedor salvo automaticamente
→ Volte ao Dashboard e veja na lista

PASSO 4: FAZER DEPLOY (Opcional)

→ Se quer testar em produção:
→ git push
→ Netlify faz deploy automático
→ Aguarde 2-3 minutos
→ Teste em seu site publicado

================================================================================
⚙️ CONFIGURAÇÃO
================================================================================

ESSENCIAL (Já feito se seguiu tutorial v1.0):
  ✅ VITE_SUPABASE_URL em .env.local
  ✅ VITE_SUPABASE_ANON_KEY em .env.local

RECOMENDADO (Opcional, para melhor experiência):
  □ Atualizar banco com SQL em ATUALIZACAO_BANCO.txt
    → Adiciona coluna "tags"
    → Melhora a busca por categoria

AVANÇADO (Opcional, para busca mais precisa):
  □ Configurar Brave Search API
    → Visite: https://api.search.brave.com/
    → Crie conta (free)
    → Copie API Key
    → Adicione em .env.local: VITE_BRAVE_API_KEY=sua_chave

IMPORTANTE: Sistema funciona 100% sem Brave!
           DuckDuckGo é completamente gratuito e sem limite.

🐛 TROUBLESHOOTING RÁPIDO
================================================================================

❌ "Aba 'Buscar' não aparece"
   → Verifique se fez npm run dev
   → Recarregue a página (F5)
   → Verifique se tem usuário logado

❌ "Nenhum resultado encontrado"
   → Tente buscas diferentes
   → Verifique sua internet
   → Recarregue (F5)

❌ "Brave não funciona"
   → É OPCIONAL! Funciona sem Brave
   → DuckDuckGo sozinho é suficiente
   → Se quer Brave: adicione API Key

❌ "Tela fica em branco"
   → Abra F12 → Console
   → Procure por erros em VERMELHO
   → Leia RESOLVER_TELA_BRANCA.txt

✅ Para mais dúvidas, leia TUTORIAL_BUSCA_AVANCADA.txt
